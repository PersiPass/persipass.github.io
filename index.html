Description

Forget about password managers with vulnerable storage. Storeless offline sync password manager allows you avoid to expose you password db. Easy to sync, easy to use.

PersiPass provides a very strong persistent password for you, based on URL, specified password size, expiration period, optional username and a secret encryption key.

The secret encryption key is not exposed to the end-user, but you can restore it by using special mnemonic sentence (a secret phrase) -- a group of easy to remember words.
